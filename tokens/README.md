# Токены дизайн-системы

Нецветовые токены фирменного стиля — сетка, тени, радиусы, переход, шрифт. Синхронизированы с `:root` боевого `dicom-design.css`.

- **[`system.css`](system.css)** — все нецветовые CSS-переменные.
- Цветовые токены — в [`../colors/`](../colors/) (`tokens.css`, `tokens.json`).

## Группы

| Группа | Токены |
|---|---|
| Шрифт | `--f-body` (Roboto) |
| Сетка | `--max`, `--gutter`, `--section-pad`, `--gap-island`, `--gutter-h`, `--r-island` |
| Тени | `--shadow-sm`, `--shadow`, `--shadow-md`, `--shadow-lg` |
| Радиусы | `--r-sm` (6px), `--r` (16px), `--r-lg` (24px) |
| Переход | `--t` (220ms cubic-bezier) |

В вёрстке используем переменные, не хардкодим значения.
