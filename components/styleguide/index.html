Компоненты DICOM TEAM

Справочник элементов сайта в одном месте. При сборке новых страниц — берёте нужные классы и применяете. Стили подключены из актуальных модулей /local/templates/dicom_v2/css/*, поэтому всё, что вы видите ниже — то же, что на боевых страницах.

С мая 2026 каждое направление услуг имеет собственный цвет. Базовый акцент сайта — синий #0D75B2, но внутри блоков с классом .dir-bitrix24 / .dir-ai / .dir-web / .dir-design переменная --c-accent переопределяется на цвет направления — и все компоненты внутри (бейджи, цифры, кнопки, ссылки) автоматически окрашиваются.

Документ закрыт noindex. Не для индексации поисковиками.

02

Модификаторы направлений (.dir-*)

Класс направления навешивается на <body> (или на любой контейнер) и локально переопределяет --c-accent, --c-accent-d, --c-accent-soft, --c-accent-rgb, --c-bg-dark. Все компоненты внутри, которые ссылаются на эти переменные, автоматически меняют цвет.

Карта URL → класс (см. header.php строки 286-303):

  • /services/bitriks24/ → .dir-bitrix24 (cyan)
  • /services/ai/ → .dir-ai (приглушённый cyan)
  • /services/App/ → .dir-web (графит)
  • /services/Design/ → .dir-design (бургунди-роуз)
Один и тот же компонент в четырёх направлениях

07

Фоны секций

Три типа фона. Чередуются для визуального ритма страницы.

section--light — белый остров, тонкая тень. Основной фон для блоков услуг, карточек, форматов.
section--gray — серый фон #F6F7F9. Для секций с белыми карточками внутри (кейсы, CTA-калькулятор), чтобы карточки «отделялись».
section--dark — тёмный фон #0F1B33. Контрастная пауза в потоке. Блок «О нас», формы, footer. Внутри .dir-* тёмная секция перекрашивается в dark-вариант цвета направления.

16

Аккордеон (FAQ)

На <details>/<summary>. Не требует JS. Иконка-плюс справа становится крестом при раскрытии.

Сколько стоит внедрение Битрикс24?

Зависит от объёма работ. Базовая настройка CRM — от 60 000 ₽, под ключ с интеграциями — от 250 000 ₽. Используйте калькулятор для предварительной оценки.

Какие есть форматы оплаты?

Time & Material (почасовая), фикс по ТЗ, недельная/месячная подписка с резервом часов, партнёрская программа с отсрочкой.

Работаете с другими CRM, кроме Битрикс24?

Основная специализация — Битрикс24 (мы официальный партнёр). Можем сделать интеграции, но новое внедрение — только Битрикс24.
