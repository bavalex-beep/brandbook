# Логотип DICOM TEAM

## Форматы

- `svg/` — векторные исходники (для веба и печати, масштабируются без потерь).
- `png/` — растровые версии (@1x, @2x) на прозрачном фоне.

Варианты: основной, монохром (чёрный), инверсия (белый для тёмного фона), favicon.

## Правила использования

- **Охранное поле** — вокруг логотипа всегда оставляем свободное пространство (минимум — высота фирменного знака).
- **Минимальный размер** — не меньше указанного, иначе теряется читаемость.
- **Не искажать** пропорции, не растягивать, не менять цвета.
- На тёмном фоне — белая (инверсная) версия, на светлом — основная.
