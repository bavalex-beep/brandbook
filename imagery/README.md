# Фотостиль и графика

Визуальный язык DICOM TEAM помимо логотипа и цвета.

- **Фотостиль** — какие изображения используем (тон, сюжеты, обработка).
- **Иконки** — стиль, толщина линий, сетка.
- **Паттерны / фоны** — фирменные графические подложки.

> Заготовка. Наполняем примерами и исходниками по мере готовности.
