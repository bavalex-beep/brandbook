# Фотостиль и графика

Визуальный язык DICOM TEAM помимо логотипа и цвета.

- **[Visual language](visual-language.md)** — правила композиции изображений: свободный центр, элементы по краям, блюр к центру, инженерный стиль, без перегруженности.
- **Иконки** — стиль, толщина линий, сетка.
- **Паттерны / фоны** — фирменные графические подложки.

> `visual-language.md` готов. Иконки и паттерны — наполняем примерами по мере готовности.
