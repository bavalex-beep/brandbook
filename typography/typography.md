# Типографика DICOM TEAM

Фирменный шрифт сайта — **Roboto**. Объявлен в `dicom-design.css`:

```css
--f-body: 'Roboto', system-ui, sans-serif;
```

## Шрифт

| Параметр | Значение |
|---|---|
| Гарнитура | **Roboto** |
| Fallback | `system-ui, sans-serif` |
| CSS-переменная | `--f-body` |
| Источник | Google Fonts |
| Лицензия | Apache License 2.0 — можно свободно использовать и распространять, в т.ч. класть файлы в этот репозиторий |

## Веса

| Вес | Применение |
|---|---|
| 400 — Regular | основной текст |
| 500 — Medium | подзаголовки, акцентные подписи |
| 600 — SemiBold | цифры секций, акценты, активные элементы |
| 700 — Bold | заголовки |

> 800 (ExtraBold) на сайте не используем — в боевом CSS он встречается единично и каноном не является.

## Подключение

**Вариант А — Google Fonts** (быстро, без файлов): см. [`webfonts.css`](webfonts.css), подключает Roboto 400/500/600/700.

**Вариант Б — self-host** (без внешних запросов): положить файлы шрифта в [`fonts/`](fonts/) и подключить через `@font-face` (см. [`fonts/README.md`](fonts/README.md)).

В вёрстке всегда обращаемся к шрифту через `var(--f-body)`, не хардкодим `'Roboto'`.

## Шкала

Размеры заголовков и текста масштабируются от базового body. Точные значения — в стайлгайде: [`../components/styleguide/index.html`](../components/styleguide/index.html) (офлайн) или `https://dicom.team/styleguide/` (когда восстановлен на сайте).

## Применение

- Не смешивать Roboto с другими гарнитурами в одном макете.
- Числовые акценты (цифры достижений, нумерация секций) — 600, цветом `var(--c-accent)`.
- Минимальный кегль основного текста — по стайлгайду, ради читаемости.
