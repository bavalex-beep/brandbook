# Шаблоны

Готовые брендированные шаблоны для повседневной работы.

| Папка | Что кладём |
|---|---|
| `presentations/` | pptx-шаблон (титул, контентные слайды, разделители) в фирменных цветах |
| `documents/` | Шапки и колонтитулы для КП и писем (docx). **Без юр. реквизитов** |
| `email-signature/` | HTML-подпись для почты |
| `social/` | Обложки, аватары, шаблоны постов для соцсетей |

> Папки — заготовки. Наполняем по мере готовности (см. CHANGELOG → TODO).
> Чувствительные документы (договоры, NDA, реквизиты) в публичный репозиторий не кладём.
